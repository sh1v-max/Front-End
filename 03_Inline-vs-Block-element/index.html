FrontEnd development

Frontend development is the development of the graphical user interface of a website, through the use of HTML, CSS, and JavaScript, so that users can view and interact with that website.

Inline vs Block element

Inline element take only required space and does not force a line break. It is used to wrap a small section of text or an image. example: span anchor tag image

Block element take full width of parent and force a line break. It is used to wrap a section of related content. example:

div

paragraph
