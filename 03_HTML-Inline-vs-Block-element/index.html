FrontEnd development

Frontend development is the development of the graphical user interface of a website, through the use of HTML, CSS, and JavaScript, so that users can view and interact with that website.

Semantic Tags

Semantic HTML elements are those that clearly describe their meaning in a human- and machine-readable way. Elements such as <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>, <figure>, <figcaption>, <address>, <time>, <data>, <details>, <summary>, <dialog> are called semantic tags.

These tags are called semantic tags because they give a meaning to the structure of a page and the relationship between elements on a page.

These tags are also called as Landmark elements

Examples of semantic tags:

  • <header> - defines the header section of a page
  • <nav> - defines a section of navigation links
  • <main> - defines the main content section of a page
  • <section> - defines a self-contained section of related content
  • <article> - defines an independent piece of content
  • <aside> - defines a piece of content that is related to the surrounding content
  • <footer> - defines the footer section of a page
  • <figure> - defines a figure with a caption
  • <figcaption> - defines the caption for a figure
  • <address> - defines the contact information for the author/owner of a document
  • <time> - defines a specific date and/or time
  • <data> - defines a machine-readable version of a piece of data
  • <details> - defines a section that contains additional information
  • <summary> - defines a summary or caption for a details element
  • <dialog> - defines a dialog box or window

em tag

The <em> tag is used to indicate that a part of text has a special meaning, such as a word or a phrase that needs to be emphasized. It is used to add a normal emphasis to the text, which is rendered in an italic font style. For example:

This is a very important message.

The <strong> tag is used to indicate that a part of text has a very important meaning, such as a word or a phrase that needs to be strongly emphasized. It is used to add a strong emphasis to the text, which is rendered in a bold font style. For example:

This is a very important message.
