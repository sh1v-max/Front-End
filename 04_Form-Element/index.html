Form in HTML

Form is a collection of elements that are used to gather information from the user. It is a container for form-related elements such as input fields, buttons, select menus, textarea, and etc.

Form is created using the <form> element. The <form> element has an action attribute which specifies the URL of the page that will process the form data when the form is submitted.

Form has two methods: GET and POST. The GET method adds the form data to the URL and the POST method sends the form data in the request body.

Form has an attribute called method which specifies the method of the form. The default value of the method attribute is GET.

Form can also have an attribute called enctype which specifies how the form data should be encoded when the form is submitted.

The <input> element is used to create a form control. The type attribute of the <input> element specifies the type of the form control. The most common types of form controls are:

  • text: A text input field
  • password: A password input field
  • email: An email input field
  • url: A URL input field
  • tel: A telephone input field
  • number: A number input field
  • date, month, week, time, datetime-local: A date and/or time input field
  • color: A color picker
  • range: A range input field
  • checkbox: A checkbox input field
  • radio: A radio input field
  • file: A file input field
  • submit: A submit button
  • reset: A reset button
  • button: A generic button

The <textarea> element is used to create a multi-line text input field.

The <select> element is used to create a dropdown list of options.

The <option> element is used to specify an option in a dropdown list.

The <label> element is used to associate a text with a form control. The for attribute of the <label> element specifies the id of the form control.

The <fieldset> element is used to group related form controls together.

The <legend> element is used to specify the caption for a <fieldset> element.
